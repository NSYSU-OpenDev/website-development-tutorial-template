# 網頁開發教學模板

這是一個幫助學生快速學習網頁設計的入門模板。

## 專案結構

```
├── index.html          # 主要 HTML 檔案
├── css/
│   └── style.css       # 樣式表
├── js/
│   └── main.js         # JavaScript 檔案
├── assets/             # 靜態資源（圖片、字型等）
└── docs/               # 文件
```

## 開始使用

1. 複製此專案到本地端
2. 用瀏覽器開啟 `index.html`
3. 開始修改並學習！

## 特色

- 簡潔的專案架構
- 清楚的資料夾分類
- 適合 HTML、CSS、JavaScript 學習與開發

## 學習目標

透過此模板，學生可以學習：

- HTML 網頁結構與語意標籤
- CSS 樣式設計與版面配置
- JavaScript 互動功能實作
